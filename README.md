# IO8 Master Repository

Central knowledge base for the IO8-lab project. This repository organises all protocols, devices, research, experiences, workflows, templates and shop references for the IO8 longevity and performance system.

## Structure

| Folder | Description |
|--------|-------------|
| `PROTOCOLS/anti-sarcopenia` | Protocols to prevent and reverse muscle loss |
| `PROTOCOLS/nervous-system-reset` | Protocols for nervous system recovery and regulation |
| `PROTOCOLS/sleep-optimization` | Evidence-based sleep improvement protocols |
| `DEVICES/heavy` | Heavy-duty equipment and usage guides |
| `DEVICES/medium` | Medium-category devices |
| `DEVICES/light` | Light and wearable devices |
| `DEVICES/nutraceuticals` | Supplement and nutraceutical references |
| `MATRICES` | Decision matrices and scoring frameworks |
| `RESEARCH` | Scientific literature and research notes |
| `EXPERIENCES` | Logged personal and community experiences |
| `WORKFLOWS` | Standard operating procedures and automation flows |
| `TEMPLATES` | Reusable document and protocol templates |
| `SHOP` | Sourcing, vendor links, and procurement notes |
